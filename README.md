# 🏥 Healthcare Analytics Platform

An end-to-end **Healthcare Data Analytics** project that transforms raw healthcare data into meaningful business insights using **Python, SQL, Statistics, and Power BI**.

The project follows a complete data analytics workflow—from data understanding and cleaning to exploratory data analysis (EDA), SQL-based business analysis, statistical analysis, and interactive dashboard creation.

---

## 📌 Project Objectives

- Understand the healthcare dataset and assess data quality.
- Clean and preprocess the dataset for analysis.
- Perform Exploratory Data Analysis (EDA) to identify trends and patterns.
- Analyze healthcare data using SQL.
- Apply statistical techniques to validate insights.
- Build an interactive Power BI dashboard for business decision-making.
- Generate actionable recommendations based on the analysis.

---

## 📊 Dataset

This project uses a healthcare dataset containing information such as:

- Patient Demographics
- Medical Conditions
- Admission Type
- Hospital Information
- Insurance Provider
- Billing Amount
- Medications
- Test Results
- Admission & Discharge Dates

> **Note:** The dataset is publicly available. Download it and place it inside the `data/raw/` folder before running the notebooks.

- Which medical conditions generate the highest billing?
